# Product Manager Alpha 0.1

Product Manager – a simple organizer for working with mock data (products)

---

## 🛒 Project Purpose

I wrote this project after a break in my studies.

---

## ✨ Features

- Viewing products
- Adding, deleting, and modifying products
- Sorting by three criteria
- Additional page with usage information (FAQ)

---

## 🛠️ Tech Stack

- **Framework:** React  

- **Bundler:** Vite  

- **Libraries:**  
    `react-redux`, `@reduxjs/toolkit`, `react-router-dom`, `axios`, 

---

## 📦 Installation

Clone the repository:

```bash

git clone git@github.com:RusikHunter/Products-Manager.git
cd products_manager
npm install
```

To run the project in development mode:

```bash

npm run dev
```

To build and preview the production version locally:

```bash

npm run build
npm run preview
```
